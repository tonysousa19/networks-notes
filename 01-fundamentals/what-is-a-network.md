# What is a Network?

## Definition

A network is a group of interconnected devices that communicate with each other and share resources.

These devices can include computers, servers, smartphones, routers, switches, printers, and other systems.

The main purpose of a network is to allow communication and data exchange between devices.

## How Networks Work

Networks work by sending data between devices using communication protocols.

When a device wants to communicate, it breaks information into smaller pieces called packets. These packets travel through the network until they reach the destination.

Each device uses addresses, such as IP addresses, to identify where data should be sent.

## Network Components

### Devices

Examples:

- Computers
- Servers
- Routers
- Switches
- Access Points

### Transmission Media

The way data travels:

- Ethernet cables
- Fiber optic cables
- Wireless connections (Wi-Fi)

### Protocols

Rules that define how devices communicate.

Examples:

- TCP
- IP
- HTTP
- DNS
- DHCP

## Types of Networks

### LAN (Local Area Network)

A network that covers a small geographic area, such as a home, school, or office.

### WAN (Wide Area Network)

A network that covers large distances, connecting multiple LANs.

The Internet is the largest example of a WAN.

### MAN (Metropolitan Area Network)

A network that covers a city or large campus.

## Why Networks Matter in Cybersecurity

Understanding networks is essential in cybersecurity because most attacks involve network communication.

Security professionals need to understand:

- How traffic moves
- How protocols work
- How devices communicate
- How attackers exploit network weaknesses

## Summary

A network allows devices to communicate and share resources. Understanding networking fundamentals is the first step toward learning system administration and cybersecurity.
